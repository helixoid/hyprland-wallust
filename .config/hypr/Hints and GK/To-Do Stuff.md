1. Use hyprpm to install hyprwinwrap plugin to hyprland and add glava to window rule to get sth like this! (Cava like visualizer at hyprland desktop)
Use Following Settings:

`plugin {
    hyprwinwrap {
        class = GLava
    }
}`
